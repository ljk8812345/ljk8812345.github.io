π与圆周运动

发表于 2019-01-17 | 分类于 永恒之眼 | | 阅读次数:
字数统计: 2k

π可能是我们最为熟知也最为重要的数学常识了，几乎存在于任何科学领域。其本身的物理意义比较明确，定义为一个圆的周长和其直径的比，这个常数本身不需要过多介绍，但它背后那个大boss——圆，值得仔细讨论。

角、角度和弧度

角(angle)

我们先从角开始讲起，因为它和圆的关系密不可分。不同阶段的教材对角有不同的定义：

(初中版)具有公共端点的两条射线组成的图形。

(高中版)一条射线绕着它的端点从一个位置旋转到另一个位置所形成的图形。

显然，从初中版的定义是静态的，无法解释超过360°的角以及负角，而高中版的定义是动态的，消除了局限性，扩大了角度的范围。更重要的是，这种动态的定义实际上引入了我们本文的主角——圆，把高中版定义中的射线换成线段，旋转一周，不就变成了圆吗？

至此，角，圆和旋转就紧密联系上了。我们先建立这样一种概念，

圆即是旋转，即圆周运动。

角度(degree)

那么角的大小如何衡量？为什么我们会用到度和弧度两种度量单位？

天文和历法是世界各农耕文明的必修课，用来记录和预测时令。人类很自然地以自我为中心来观察宇宙。古人对星象的长期观测发现，各星象在夜空中的位置每天都在移动，斗转星移，直到一个周期——也就是一年后，又恢复原位。

为了记录星象，人们按天把一个星象运动周期分为了360度，尽管古人已经能得出一年365天，但是360由于因数众多，数学计算方便得多。因此，圆周被分为了360度，这种度量角大小的方式就是角度制，它很容易被人理解和接受，正如人类有十个手指从而发明了十进制一样。但它却带有相当的主观性，假如人类生活在火星（公转周期687天）上，那么圆周可能就是680度了。

弧度(radian)

角度在数学中不仅不够严谨，还会让诸多公式变得繁琐复杂，于是就引入了新的度量制——弧度。前面说过，角和圆周运动密不可分，从圆周运动的角度理解：

角度是从观察者的视角来看待圆周运动的，根据自己转头的程度来衡量“角”。

弧度是从运动者的视角来看待圆周运动的，根据圆周运动者的移动距离来衡量“角”。

也就是说，弧度从某种意义上讲，代表了一个点在圆周上运动的“距离”。但是圆周运动的绝对距离并没什么意义，我们对这个“距离”归一化使得弧度在不同大小的圆中意义统一，就得到了弧度的定义：

弧度=弧长(距离)/半径(距离)

由圆周率π的定义可知，一个圆就被划分为了2π个弧度($2\pi r/r$)，也就是说，弧度是一个纯数字，是个无量纲的实数。

这个特性很重要，我们很难理解用角度制表示的三角函数，自变量是以“度”为单位的数，因变量却是一个无单位的数。而有了弧度，我们就可以在实数坐标系下研究三角函数了，三角函数的定义域和值域都统一在了实数域下，三角函数在这个基础上才能发展，例如$arcsin0.5$就是一个弧度制表示的实数，这在角度值下就很难理解。

我们可能都经历过两种角制转换时的别扭，都混淆过$sin1°$和$sin1$意义，我觉得这是基础教育中存在的问题。由于文化的习惯性，角度制在生活中还有些意义，但是对于现代科学，直接从圆的概念引入弧度制和弧度制意义下的角才是正确的学习路径。

我们再建立这样的概念，

“角”是单位圆上的一个“长度/距离”。

这样，我们再去看“角速度”，就容易理解的多了，“rad/s”是不是就跟“m/s”一样了？

三角函数

三角函数的起源和发展也离不开天文学，它是一种对天文观察结果进行推算的方法，用于间接测量距离和长度。

古希腊科学家泰勒斯利用日影测金字塔高度的故事被广泛熟知，他用的是最简单的等腰三角形测距。在一天中的某个时刻，人的身高与影子等长，金字塔的高度与其影子等长，这就是三角函数$tan\pi/4$的基本应用。

教材中三角函数的定义方法也有高低级两个版本：

直角三角形定义法：在直角三角形中，任意一锐角∠A的正弦是对边与斜边之比，∠A的余弦是临边与斜边之比，∠A的正切是对边与临边之比。

单位圆定义法：设α是一个任意角，它的终边与单位圆交于点$P(x，y)$，那么：$y$叫做$α$的正弦，$x$叫做$α$的余弦，$y/x(x≠0)$叫做$α$的正切。

直角三角形定义法是人们最初开始研究三角函数时的定义，仅限于锐角，目的在于研究三角形的各种各样的几何量之间的函数关系，我们可以称之为“三角学”。

而单位圆定义法下的三角函数，不仅将锐角扩展到了任意角，更体现了它最重要的特性：周期性，这使得三角函数成为描述周期现象最重要的数学模型，我们要讲信号，通信就离不开周期振动，离不开代表周期现象的三角函数和圆，他们几乎是一切现代科技的基础。从“边长之比”的角度来定义三角函数，比值的周期性变化规律需要经过推理才能得到，显然无法体现这个精髓。
