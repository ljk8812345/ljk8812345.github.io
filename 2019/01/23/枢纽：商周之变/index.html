《枢纽》拆书记——商周之变

发表于 2019-01-23 | 分类于 达拉然 | | 阅读次数:
字数统计: 2.3k

施展老师的《枢纽》是我去年读过的最有价值的一本书，一部史学著作，它站在更高的时空维度来解读中国历史，从中原、草原及海洋相互塑造的角度出发，论证现代中国的历史定位——“枢纽”，比流水账式的叙事史有趣的多。书中诸多奇妙的观点可能需要我写数篇文章来整理和消化。

中国历史上发生的第一次重要转型，是商周之变，发展出了“中国”的概念，催生了中华文明最初的普世主义理想，经历了封建社会的兴起和终结，以及思想大爆发。

“中国”——文明的中心

周朝第一次出现了“中国”这个词，指的是“中央之国，天下中心”，但这个中心不是地理意义上的中心，而是文明意义上的中心。周武王说要“宅兹中国”，是说要住到最文明发达之地。

就是说“中国”是文明水准最高的地方，而不是某块具体的土地。在马关条约签署的时候，伊藤博文就曾拒绝清朝官员在条约上写中国二字，只接受写清国。他认为落后的满清没资格代表中国，日本才能配得上“中国”二字。可见到了20世纪，东亚文明圈还是从文化的角度来理解中国。

古代中国认为文明并不属于某个特定的族群，而是属于天下所有人。“中国入夷狄则夷狄之，夷狄入中国则中国之”，华夏和蛮夷并不是一个横向的血缘问题，而是一个纵向的文明程度问题。

天命观之变——普世理想的产生

中华文明这种开放的“天下主义”的普世价值观是如何产生的呢？这和古人天命观的变化有很大关系。

“天命降于商人”

殷商一朝，商部落的人才算是人，外族人是奴隶，是牲畜，是人祭，总之，不算是人。商人只是想维持自己的优势地位，站在了多数人的对立面，是民族主义。

“天命降于周王”

周人看似变得狭隘的世界观实则是变得更加普世。周王不仅是周人的王，更是天下共主，不仅要照顾周人的利益，也要顾及天下人的利益，要为所有人主持正义。周人开辟了一种“普天之下，率土之滨”的秩序，奠定了中华文明基础的世界观和价值观，普世主义理想由此产生。

“封建社会”的兴起和终结

我们一直以来都认为秦到清末这两千多年时间为封建社会，这是机械套用了马克思社会发展理论的而得到的错误结论，真正意义上的封建社会只存在于周朝。秦以后的郡县制帝国形态在西方历史上不曾出现，而中国也没有西方意义上的奴隶社会，因此马恩的理论体系下的社会形态划分不能套用于中国历史。

“封建亲戚，以藩屏周”，周天子将土地分封给亲戚功臣建立诸侯国，让这些国家作为藩篱拱卫周朝。“封建”也巩固了周天子天下共主的地位，通过诸侯国的间接管理，有效拓展了统治区域和周文化的影响范围；但这也是不得已而为之，因为当时的生产力和政治制度水平不足以支撑一个能有效统治全国的庞大中央政府。

诸侯们都有比较强的独立性，周天子无权干涉诸侯国内部事务，他们之间更像是合伙人关系。随着诸侯间的征伐吞并，出现了比周天子更强大的诸侯国，天子权力衰落，沦落成普通诸侯，封建社会也开始终结。这个过程中，社会秩序极大地动荡，也诞生了璀璨的精神文明。

秩序失衡

  1. 战争秩序——礼仪性到实用性

    西周时期，战争是件高贵的事，是讲“礼”的。“国之大事，唯祀与戎”，只有贵族才有资格参战，目的是恢复“礼”所代表的社会秩序，战争规模和杀伤性都很小。

    春秋战国时期，以楚国为代表的“蛮夷之邦”不再按中原“礼”的套路出牌，礼仪性的”队列比赛”变成你死我活的生死搏斗，与此同时，平民加入的战争使得“人口”成为了资源，杀人也成了战略目的，战争的实用性超越了礼仪性。

  2. 君臣秩序——平民力量的崛起

    西周时期，为什么战争是贵族的事而底层平民无法被发动呢？这是封建社会的多级分封结构导致的。周天子无法统治广袤的土地而分封诸侯，诸侯随着人口和领土的扩张，也需要再分封自己的亲戚近臣做次级诸侯，例如晋国的韩赵魏三家都是次级诸侯。中世纪的欧洲有这么一句话：“我封臣的封臣不是我的封臣”，正如周天子无法管控诸侯一样，诸侯们也不能完全控制次级诸侯，实际上能发动起来参战的人，仅限于自己城中的直系部队。

    春秋末战国初，两个重要的技术变迁打破了君主(诸侯)和贵族(次级诸侯)的力量均衡。

  • 铁器和牛耕技术——农业生产效率极大提高

    过去耕作效率低下，平民被掌握生产资料的贵族束缚，只有精力耕种公田，无力开垦私田；但是生产效率提升之后，平民有余力开垦私田，甚至摆脱贵族束缚。

    平民逃离井田会削弱贵族的实力，这是君主所乐见的，于是君主和平民间建立了直接的联系——君主向耕种私田的平民提供保护，平民直接向君主交税。贵族所控制的井田制逐渐衰弱，君主优势扩大。也有贵族(次级诸侯)先承认并保护耕种私田的平民，将他们纳入自己的统治，最后力量越来越强大取代了君主成为诸侯，如韩赵魏，田氏代齐等等。

    总之，平民的力量开始崛起，谁控制了平民，谁就具有了争霸天下的实力。

  • 竹简书写技术——知识传播成本极大降低

    周朝前期，文字是刻在金属器具上，掌握在巫师为代表的贵族手中，平民没有机会识字和获得知识，君主必须依靠贵族来处理政务。竹简的出现使知识的传播成本大为降低，平民有机会获得知识，出现了大量有文化、无财产的“游士”。

    和贵族不同，这群游士无根基，无法对抗君主；也没有独立性，地位和荣耀都取决于君主的赏赐，是君主的附庸而不是合伙人。游士和君主抛开贵族一拍即合，前者获得了超越贵族的权利和地位，后者有了私人班底来实现自己的意志。

    诸侯国从君主贵族共治的结构转向了君主独裁的官僚制结构，这个过程是通过变法实现的。随着贵族阶层被肃清，在没有能约束君主的力量，封建制也就走向了终结。

思想变革

秩序失衡的时代，往往是思想迸发的时代。人们长久以来形成的三观崩塌了，开始寻找新的理解世界的方式，催生了诸子百家。各种学派从根本上都要回答一个问题，如何对待传统？

  • 回归传统——复古——儒家

    儒家认为，礼崩乐坏都是因为人们背离了传统。

    “古”就是“礼”所代表的秩序，而这套“礼”记录和掌握在巫师阶层。因此，孔子也可认为是巫师阶层的集大成者，他认为复古的方法是“克己复礼”，克制自己的欲望冲动，按礼的规矩做事。在动荡的年代，这么做显得很迂腐，是需要勇气充满担当的，因此儒家侧重对道德和勇气的赞美。

  • 抛弃传统——开新——法家

    法家认为，传统已经过时了，要开创新规范。

    “法”是君主的意志，官僚是执法者。法家三宝“法，术，势”，推崇用诡诈之术来控制人，懂得利用和创造对自己有利的局势，既受君主喜爱，也被君主忌惮。

  • 不在意传统——出世——道家

    道家认为，你们管的事情太多才把世界搞坏了，传统好不好、有没有用，根本不重要。顺其自然，让世界让百姓自由自主的活动不加干涉，问题自然就解决了，这就是“无为而治”。
